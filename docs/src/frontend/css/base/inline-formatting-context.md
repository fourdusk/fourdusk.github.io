## IFC 的特性

### 内部元素一行排列

IFC 中的元素从包含块顶端水平方向逐一排列，水平方向的 margin、border、padding 生效

### 内部元素自动计算宽度和高度

IFC 中的元素的宽度和高度是根据其内容自动计算的，且不支持设置宽度和高度

## IFC 的应用场景

### 1. 行内元素的布局

IFC 可以用于行内元素的布局，例如文本、图像等

### 2. 导航菜单

IFC 可以用于导航菜单的布局，例如水平导航菜单

### 3. 表单元素

IFC 可以用于表单元素的布局，例如文本输入框、按钮等。
